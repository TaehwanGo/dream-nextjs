## 9.2 프로젝트 셋업

```bash
npx create-next-app@latest
```

- 선택한 옵션들

  - app name: blog
  - use typescript: yes
  - use eslint: yes
  - use tailwindcss: yes
  - use src directory: yes
  - use custom alias: no

- tailwindcss 설치를 수동으로도 할 수 있다
  - 공식 문서 참고
    - https://nextjs.org/docs/app/building-your-application/styling/tailwind-css
